# WordPress Law Firm theme development from scratch

The theme was built with Custom Post Types and Customizer UI.
All PHP files was organized by template hierarchy.
Models and template parts are selected in a separate layer.

## Requirements

> PHP 7+
> WordPress 5+

### Plugins

> Advanced Custom Fields
> Advanced Custom Fields: Repeater Field
> Contact Form 7
> Yoast SEO

## Usage

Anyone can use this theme as a starter kit.
